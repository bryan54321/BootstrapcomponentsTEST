Nintendo 64

the Nintendo 64 was the first console that made 3D graphics.
price=$199
the controller is used by connecting a cord into the console

The Nintendo 64 came out inn June 23, 1996 in Japan, the original cost of the console was $200. The console has a slot to put the game that you wanna play, it has a power button to turn on and off your console it has a reset button, it has four connnections so that you can plug in four controllers and play, it also has a memory expansion. The controller has a d-pad, has four yellow button on the left, under those there are two buttons, an A and a b button, in the middle there is a start button allowing you to start or pause the game, and under the start button there Is now a joystick which is used to play some games. In the back of the controllers there are two buttons a L and R button, these buttons are used to help you in a game and to complete it. In the bottom of the controller there is another button which can also be used in game. The controller has a cord coming out of the controller so that you plug it into the console. some of the first games that came out were super Mario 64, Pilotwings 64 and saikyo habu shogi. Later on, other games came out like super smash bros., paper Mario, Zelda, donkey Kong 64, resident evil 2, Mario cart, 64 and lots more. In the first day of the Nintendo 64 sold 300,000 units on the first day of its release.
